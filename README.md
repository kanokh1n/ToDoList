# ToDoList API

Проект состоит из трёх сервисов:

- **API Service** (`:8080`) - REST API прокси, принимает запросы от клиентов
- **DB Service** (`:8081`) - бизнес-логика и работа с базой данных
- **PostgreSQL** (`:5432`) - база данных

## Бастрый старт 
### Клонируем репозиторий 
git clone 

### Переименовываем файл `.env.example` -> `.env`, если надо, то меняем значения переменных в нем 

### Собираем проект
    make build 

### Запускаем контейнеры
    make up

### Выполняем миграции 
    make migrate-up

### Проверяем что все рабоает 
    curl http://localhost:8080/tasks
    #должен вернуть []
